## 2. 📒 字面量（Literal）

- 字面量在 JavaScript 中表示值。
- 字面量是固定的值，不是变量。
- 你可以在脚本中直接书写字面量。
- 字面量可以是各种数据类型，包括数值、字符串、布尔值、对象、数组等。
